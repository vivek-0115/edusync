# EduSync

**EduSync** is a modern, scalable School Management System (ERP) designed to fully digitalize school operations. It integrates students, teachers, parents, and administrators into a unified platform with real-time synchronization and smart analytics.

---

## Tech Stack

### Frontend

* Vue 3 (Composition API)
* Pinia (State Management)
* Tailwind CSS

### Backend

* FastAPI (Python)
* JWT Authentication
* Role-Based Access Control (RBAC)

### Database

* PostgreSQL

---

## Features

### Admin

* Dashboard & analytics
* User management (students, teachers, parents)
* Class & timetable management
* Fee management
* Reports & insights

### Teacher

* Attendance management
* Assignment upload & grading
* Student performance tracking
* Content sharing

### Student

* View timetable & attendance
* Submit assignments
* Access learning materials
* View results

### Parent

* Monitor student performance
* Fee tracking & payments
* Communication with teachers

---

## Authentication & Security

* JWT-based authentication
* Role-based access control
* Secure password hashing (bcrypt)
